Vue.js ecshop 9.Loading Overlay 讀取效果

2024-08-22
#Vue.js / ecshop

在我們的後台頁面稍微作出個樣子之後，可以稍微來新增一些樣式，讓整個畫面看起來更專業！
此篇文章想跟大家分享的是 讀取的視覺效果 Loading，適當加上視覺效果可以讓頁面在讀取時不會一片空白，若是讀取時間較長，也可以透過讀取效果來提升使用者體驗，那我們就開始吧！

使用 vue3-loading-overlay 套件

  • 安裝套件（參考文件）
  • 將元件新增到 main.js
  • 回到 ProductsPage.vue 插入元件 參考 文件 的撰寫方式： * 接著將 `isLoading` 這個狀態加到 `data`，並預設為 `false`

自訂樣式

若是希望可以自訂樣式，可以參考 loading.io 此網站提供許多樣式可以做使用。

  • 此網站會需要先註冊並登入方可下載，有些資源是免費的，有些是付費的，可以斟酌使用
  • 找到喜歡的讀取效果後，點擊下方的 Download As > more > css 來進行下載
  • 下載後會得到一個 HTML 檔案：
    • 請將其中的 HTML 內容新增到 <LoadingOverlay></LoadingOverlay> 裡面
    • 接著同步新增文件中的 CSS 內容到你的檔案中即可做使用

2024-08-21
#Vue.js / ecshop

先前在 Vue.js ecshop 專案 6.props & emit 資料傳遞 文章中有分享過 新增商品 的功能，此篇文章要補充其中 上傳圖片 的功能，那我們就開始吧！

透過 API 上傳圖片

  • 在 ProductModal.vue 下方這段程式碼新增 ref、@change 片段
  • 依照 API 提供的上傳圖片格式 multipart/form-data 新增上傳事件 * 新增 `uploadFile()` 事件 * 其中使用 `new FormData()` 來建立一個 `form-data` 格式的內容 * 接著透過 `formData.append('欄位名稱', 上傳的資料)` 來增加欄位到表單中 * 再透過上傳檔案的 API 做推送
  • 圖片成功上傳後，在 console 可以看到 data 裡面會有 imageUrl
  • 最後再加上判斷式 讓圖片 imageUrl 儲存到 tempProduct

Vue.js ecshop 2.匯入 Bootstrap 5

若希望在專案中使用 Bootstrap 5（以下簡稱 BS5） 套件有兩種方式：CDN 以及 npm 安裝。
此篇文章，首先會與大家分享如何透過 npm 安裝 Bootstrap 套件，並且自訂樣式設定；接著也會補充透過 CDN 的安裝方式，那我們就開始吧！

使用 NPM 安裝 Bootstrap

於終端機輸入以下代碼進行套件安裝：npm install bootstrap

調整 App.vue

  • 安裝完畢後，接著進入 BS5 六角學院翻譯文件 ，找到左側 自訂義 > Sass
  • 將上述欄位的文件插入到 App.vue 檔案的 <style> 中
  • 接著就可以開始新增 Bootstrap 元件到專案中

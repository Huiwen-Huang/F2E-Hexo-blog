Vue.js ecshop 3.製作後台登入頁面

2024-08-12
#Vue.js / ecshop

此篇文章正式進到此專案的第一個頁面：後台管理登入頁面！
在此專案中我們將會透過 Axios 套件 來串接 API，那我們就開始吧！

安裝 Vue Asiox

  • 於終端機輸入指令進行安裝： npm install --save axios vue-axios
  • 接著回到 main.js 新增以下程式碼片段 (資料來源：官方文件)

新增登入頁面

  • 首先在 src > views 中新增 LoginPage.vue 檔案
  • 製作登入頁面表格 <form>
  • 接著新增 <script> 內容：
    • data 中要有 API 文件所需參數 username、password
    • 且建立一個 signIn() 用來登入使用
  • <template> 的部分主要針對 <form> 、 <input> 新增相關片段：
    • <form> 加上 @submit.prevent="signIn" 點擊觸發登入功能
    • <input> 針對各欄位加上 v-model 對應到各資料名稱
  • 完成後實際使用 註冊 Vue API 的帳密 進行登入測試
  • 登入成功，可以在 console 看到系統回傳 登入成功 訊息

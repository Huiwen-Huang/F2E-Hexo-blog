前情提要

這是艾瑪在上完六角的 Vue3 課程後，迎來的第一個挑戰：電商網站。
在這次的專案中，老師有提供 API 文件讓我們進行資料抓取，由於這部分屬於六角學院的資產，在筆記中就不會特別分享，還請見諒。

建立 Vue Cli 專案

安裝參考官方文件

  • 於終端機輸入 npm install -g @vue/cli 進行安裝
  • 若要查看版本號可輸入 vue -V ，同時可以確認是否有安裝成功

創建專案參考官方文件

  • 於終端機輸入 vue create + 專案名稱 即可建立新的專案

  • 接著需完成此專案所需的相關設定

    • 選擇 Manually select features 手動進行設定
    • 此專案會使用到 Vue Router 所以要選擇 Router

    圖片來源：Vue cli 創建官方文件

各資料夾用途

  • /node_modules：用以存放套件相關文件
  • /public：主要放純靜態檔，當資源連到檔案中，不需要經過編譯
  • /src：需要經過編譯的檔案，通常會經過壓縮

如何區分資料放置位置

  • /public
    • 主視覺圖片檔（對畫質很刁鑽）
    • 外部 UI CSS
  • /src
    • 外部 libs JS
    • 自己寫的 SCSS

進入開發模式

  • 於終端機輸入 npm run serve
  • 可以看到 Local、Network 兩組網址
    • 個人開發可直接進入 Local： http://localhost:8080/
    • 小組開發可點擊 Network： http://192.168.0.101:8080/
    • 進到 http://localhost:8080/ 看到下方畫面，表示已經完成 Vue 專案的建立

2024-09-06
#Vue.js / ecshop

完成商品頁面後，接下來我們就要進到將商品 加入購物車 的環節啦！
此篇文章將會與大家分享，加入購物車可以搭配效果做樣式的調整，以及購物車頁面的製作，那我們就開始吧！

加入購物車

  • 首先我們需要在 ShopProdcutsList.vue 新增 addCart(id) 函式，並帶入商品 ID
    • JS
    • HTML
  • 若希望讓顧客在點擊加入購物車後有相關樣式，可以將被點擊的按鈕，在打 API 時進行 disabled 樣式呈現
    • 新增 status: { loadingItem: '' }
    • JS
    • HTML
  • 設定成功後，點擊按鈕會有反灰的效果，ajax 完成後就會恢復正常

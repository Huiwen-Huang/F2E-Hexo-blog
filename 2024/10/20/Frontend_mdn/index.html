Frontend 前端初學 - 線上工具分享 (MDN)

2024-10-20
#線上工具分享

今天想來跟大家分享我自己在學習前端路上，覺得對初學者蠻有幫助的三個學習網站。
分別是 **MDN、W3Schools、freeCodeCamp**，相信大家應該都不陌生，那就讓我帶你們進一步了解這三個網站的資源可以如何使用吧！

MDN

首先要跟大家分享的是最常用到的 MDN 技術文件，不過大家一定很好奇不就是參考怎麼寫就好了，有什麼好特別拿出來說的？
正是因為我自己在參考 MDN 文件，也是過了很久之後才發現原來看技術文件有幾個重點步驟，且很多資訊都是很實用的！

我們以 Array.prototype.splice() 這個方法為例，參考文件：

  1. 嘗試一下 （實際使用）
    每個功能介紹頁面，第一個區塊「嘗試一下」就是可以實際操作的區塊。
    在這個區塊，除了可以點擊下方的 Run 按鈕產生預設內容的結果外，也可以直接編輯程式碼區塊進一步做測試使用唷！

  2. 語法：參數、回傳值

  • 參數
    接下來針對每個功能的使用方式也會有更近一步的說明。
    像是其中的參數，提到起始值、刪除元素數量、要新增到陣列的元素等等，我們可以藉此了解每個功能要放的參數含義，以及一些限制與變化等等。
  • 回傳值
    接著會看到一個我覺得蠻重要的資訊：回傳值
    光是這邊舉例的 陣列方法 ，每種方法的回傳值都不盡相同，像是 .splice() 與 .slice() 的回傳值就不一樣！在使用前都可以多參考 回傳值 這個區塊的資訊唷！
  1. 瀏覽器相容性
    最後要分享的也是最重要的！
    無論是 HTML / CSS / JavaScript 在使用任一功能之前，如果不是大眾經常使用的，建議可以上來參考 瀏覽器相容性 的部分，有些功能在特定瀏覽器可能會有不支援的情況！
    如果沒有事先注意到，功能上線後才發現在特定瀏覽器無法正常顯示或執行，要再回頭調整可能會需要花更多的時間，這也是我要跟大家一起共勉之的部分！

剛好提到瀏覽器相容性，這邊也跟大家分享另一個專門查詢瀏覽器相容性的網站: Can I use
Can I use 提供更詳細的瀏覽器個版本支援的情況，也提供給大家參考！

今天的分享就到這邊，歡迎大家繼續閱讀 Frontend 前端初學 - 線上工具分享 (W3Schools) 參考我幫大家整理的使用方法吧！
